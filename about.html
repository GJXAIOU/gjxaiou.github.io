Hey，我是高基旭(GJXAIOU)，来自南京邮电大学，研究僧一枚。

这是我的利用 GitHub Pages 与 Jekyll 搭建的 个人博客。我在GitHub主页👉GitHub·GJXAIOU 与 CSDN 主页👉GJXAIOU。如果有什么问题，欢迎提出探讨~

目前已经完成了 Java 基础知识、Java Web 框架（SSM）、Java 虚拟机、Java 并发、数据结构与算法、数据库、设计模式、计算机网络、Python 等知识的学习。并且完成了自己的第一个项目：O2O。

技术学习是一条很漫长的道路，也没有捷径。一年多以来每天都充满着热情，无论是面对满屏 bug 时在 StackOverflow 上寻找解决之道，还是 Debug 好久程序终于运行成功，哪怕是用自己学习的算法知识 AC 了一道题目，都让自己忘乎所以的投入其中。一年前还是懵懂，一年后已经渐入佳境，多年后相信会头发日渐稀疏，哈哈哈哈。

现在的自己一方面不断的加深自己的技术栈，同时尽自己所能的反哺社区，开源了自己几年来各个方面学习的几十万字笔记，开始在博客平台上分享自己整理的知识以及问题解决方案，搭建自己的博客，加入掘金翻译社区参与优秀外文的翻译以及校对，搭建师门知识分享平台。。。 虽然所学知识尚浅，但是我始终认为每个人都有自己的发光点，在算术之外 1 + 1 永远大于 2， 我愿意并将不懈努力的的放大那个 1。
